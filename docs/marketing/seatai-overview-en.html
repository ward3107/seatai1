The everyday problem

Seating charts are slow, manual, and easy to get wrong

⏳ Doing it by hand today

  • Teachers spend 30–60+ minutes per class shuffling names on paper or sticky notes.
  • Juggling who-can’t-sit-with-whom, friendships, eyesight and focus needs is mental overload.
  • Special-needs requirements (front row, quiet zone, mobility, EAL/ESL) get missed under pressure.
  • Every new term, new student, or behaviour issue means starting over.
  • Charts live in one teacher’s head — hard to hand to a substitute.

✅ With SeatAI

  • A balanced, rule-respecting plan in seconds, not an evening.
  • Every constraint is enforced automatically and nothing is forgotten.
  • Each placement comes with a plain-language reason teachers can stand behind.
  • One click to re-shuffle, rotate, or adjust when things change.
  • Print an anonymised chart for a substitute in one tap.
How it works

From class list to optimised chart in four steps

1

Add the class

Type students, import a CSV, or load a sample class to try it instantly.

2

Set the room & rules

Pick a layout and add any rules — keep apart, front row, peer mentors.

3

Optimise

A genetic algorithm searches thousands of arrangements for the best fit.

4

Review & print

Fine-tune by drag-and-drop, see why each seat was chosen, export & print.
